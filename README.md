## 🌟 Top Features
1. **High-Security Geofencing:** Auto-attendance marking only when students/employees are within the allowed physical radius.
2. **Dynamic Secure QR Code:** Temporary, encrypted QR codes that change automatically, preventing fake or proxy attendance.
3. **Cross-Platform Dashboard:** A sleek, fully responsive React.js Admin Panel to manage students, view daily attendance summaries, and download reports.
4. **Offline QR Scanner Mode:** In-built web scanner for administrators to mark attendance if students do not have GPS enabled.
5. **Modern Android App:** Built natively in Kotlin with Jetpack Compose for a blazing-fast user experience.

## 🛠️ Technology Stack
* **Backend:** Laravel (PHP), MySQL
* **Frontend Admin:** React.js, Tailwind CSS, Vite
* **Mobile App:** Android Native (Kotlin, Jetpack Compose)
* **Integrations:** Google Location Services, ZXing QR Library
